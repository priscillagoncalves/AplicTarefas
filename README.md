• AcessoBD - Classe: AcessoDados.cs

• DTO - Classes: Tarefa.cs e TarefaCollection.cs

• Negocios - Classe: TarefaNegocios.cs

• Interface - Classes: FrmMenu.cs, FrmCadastrar.cs e AcaoTela.cs
